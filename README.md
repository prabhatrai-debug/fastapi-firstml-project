# FastAPI First ML Project

This project is a basic Machine Learning prediction API built using FastAPI.

## Features
- FastAPI backend
- Machine Learning model integration
- Real-time prediction API

## How to Run

1. Install dependencies:
```bash
pip install -r requirements.txt
```
2.Run the FastAPI server:
